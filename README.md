## 琴男主钟离欧洛伦夹门火刀

### 简介
此为**BetterGI**脚本，修改自up主**乱舞の菜刀**，原视频：https://www.bilibili.com/video/BV16mz4BkERN

原视频使用女主和提纳里，用了脚本过了大号，在此十分感谢up主，但是我小号是是男主，而且没有提纳里，就自己修改了一下，欧洛伦e的后腿距离比提纳里大，所以钟离e前要加一个前进，又由于该脚本下，前进后男主的荒星会偏右，所以加上了个向右小幅度偏转，成功率在我电脑上比较高。

由于有视角的偏转，请自己调整分辨率到1920×1080，游戏设置中所有灵敏度都调成3，或者自己在脚本设置中自己微调。
